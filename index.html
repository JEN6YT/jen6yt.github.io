In-Context Algorithm Emulation with One Frozen Attention Layer @ MAGICS, Northwestern University

Jerry Yao-Chieh Hu*, Hude Liu*, Jennifer Yuntong Zhang*, Han Liu
Neurips 2025.

pdf code

We show that a frozen Softmax Attention Layer can emulate Attention, gradient descent, Linear Regression, LASSO, and Ridge. We mathematically prove that with attention layer mimick properties of linear functions. We empirically demonstrate that approximation accuracy increases with an increasing number of attnetion heads. Our result provides a fundamental theory of a minimalist Transformer structure for internal algorithm swapping, demonstrating the ability of frozen Transformer emulating algorithms.

Deep Learning of Structured and Continuous Policies for Aggregated Heterogeneous Treatment Effects

with Dr. Will Y. Zou and Shuyang Du

Jennifer Yuntong Zhang, Shuyang Du, Will Y. Zou

pdf code

Existing work on HTE adapts well on binary treatment action space (treated or not), lacking continuous treatment policy where treatment intensities play a crutial role. In this work, we propose a novel deep learning framework to learn structured and continuous treatment policies for HTE estimation. We construct a Neural-Augmented Naive Bayes layer within a deep learning framework to incorporate an arbitrary number of factors that satisfies the Naive Bayes assumption. The factored layer is then applied with continuous treatment variables, treatment assignment, and direct ranking of aggregated treatment effect functions. Together, these algorithms build towards a generic framework for deep learning of heterogeneous treatment policies, and we show their power to improve performance with public datasets.

Hebbian Learning and Local Inhibition @ Social AI Lab, UofT

with Prof. William Cunningham

This project investigats biologically plausible neural network architecture. Inspired by Hebbian rule, Locally Competitive Algorithm, predictive coding, we develop activation and weight update rule based on minimizing a pre-defined "energy" function (i.e. aggrgated reconstruction error of the network for all layers with top-down feedback for later layers). We evaluate its performance on standard benchmarks such as MNIST, Cifar10, and ImageNet, and it achieves SOTA performance. In addition, we deploy visualizaiton techniques to verify model interpretability and biologically plausibility.

Methane Emission Drivers and Baseline Calculations @ Risk Lab, UofT

Yuntong Zhang, Behzad Azadie Faraz, Luis Seco
In ADIA Lab.

pdf code

This study presents a comprehensive analysis of feature importance selection methods aimed at identifying the key drivers of methane emissions globally. We identify 8 critical features among 58 contributing most to methane emissions. In addition, we apply 4 machine learning models, Support Vector Machine, LightGBM, XGBoost, and Random Forest, to estimate methane level across various countries. Moreover, we determine and anlayze the local and global trend for methane emission across 220 countries and identify the top countries that have the steadiest and the most decrease as well the most increase in methane emission in the recent 30 years.
